# Healthcare-project

# Mini Healthcare Support Web App

Live Demo:
https://saiiteja.github.io/Healthcare-project/

## Overview
A concept-level healthcare support web application designed for NGOs and healthcare organizations to manage support requests using AI and automation.

## Features
- Patient support, volunteer registration, and general inquiry forms
- AI-powered chatbot for FAQs and basic healthcare guidance (Chatbase)
- Automated email notifications on form submission using EmailJS
- Fully frontend-based application with no backend dependency

## AI / Automation
The integrated Chatbase chatbot helps users understand how the website works, provides basic emergency care information, and guides them to the appropriate support form. Complex or unclear requests are escalated to human support via email.

## Tech Stack
- HTML, CSS, JavaScript
- EmailJS for client-side email automation
- Chatbase for AI chatbot integration
- GitHub Pages for hosting

## Use Case
Suitable for NGOs and healthcare initiatives to receive requests efficiently, reduce response time, and provide quick assistance to users.
